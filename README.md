# April's Portfolio
Hi! This is just a portfolio website I made. It's very simple, and it was made as an easy way to show people some stuff I've made in the past. 
It's a single-page site with pretty normal sections, such as an About Me section, a Services section, and a grid of my projects.

## How this was made
It was made using HTML, CSS, and JS. It is hosted on GitHub Pages.
I originally made the website ages ago (maybe 2 or 3 years ago?) back when I wanted a new portfolio website. I only recently decided to just update the information and add a few new animations and features. I have to update it anyway, since Hack the North applications are out... but I also didn't want to make an entirely new one, so I'm updating this one
I haven't run into many issues recently with it, but I remember I originally wanted the "Contact Me" section to send an email, but I ran into too many problems, so I just had it edit a Google Sheet.

## How to run the project

1. Clone the source code to your device
   ```sh
   git clone https://github.com/ohi-711/ohi-711.github.io.git
   ```

2. Open it directly in your browser
   ```sh
   open index.html
   ```
