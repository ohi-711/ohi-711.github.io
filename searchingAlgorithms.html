SEARCHING ALGORITHMS

LINEAR SEARCH

Linear search or sequential search is slow, but it works on any array. It interates through the array until the current element is equal to the element being searched for. Then, it returns the index of said element.

BINARY SEARCH

Binary search involves repeatedly cutting an array in half until the element is found. If the middle element is greater than the target element, the top half is discarded. On the other hand, if the middle element is smaller than the target element, the bottom half is discarded. This method is faster than a linear search. However, it only works on a sorted array.
