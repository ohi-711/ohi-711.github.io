SUBCLASSES AND INHERITANCE

SUBCLASSES

THe principle of inheritance is implemented in Java through subclasses. A subclass has the same fields and methods as the class it is inheriting from. The class being inherited from is called the super class. The extends keyword is used in the class definition to write a subclass in Java. In order to access methods from the super class, the super keyword is used to call these methods. This is commonly used in the class constructor.
